# 多吃点的小饭桌

多吃点的个人博客——技术、折腾与日常

🌐 **[blog.zkl2333.com](https://blog.zkl2333.com/)**

## 技术栈

基于 [Astro Cactus](https://github.com/chrismwilliams/astro-theme-cactus) 主题构建，Astro v6 + Tailwind v4。

## 本地开发

```bash
pnpm install
pnpm dev        # localhost:3000
pnpm build && pnpm postbuild
pnpm preview
```

## 内容目录

```
src/content/
├── post/   # 博客文章
├── note/   # 碎念
└── tag/    # 标签说明页（可选）
```
